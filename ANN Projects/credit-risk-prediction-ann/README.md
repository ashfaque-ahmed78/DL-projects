# 📌 Project Overview

This project focuses on **Credit Risk Prediction**, a critical problem in the banking and financial industry.

Using customer financial information, the model predicts whether a customer is likely to be a **high-risk borrower** or a **low-risk borrower**.

The project applies **Artificial Neural Networks (ANN)** to identify hidden patterns within customer credit data and assist financial institutions in making informed lending decisions.

---

# 🎯 Objectives

The primary objectives of this project are:

✅ Analyze customer credit information

✅ Perform Exploratory Data Analysis (EDA)

✅ Identify risk-related patterns

✅ Build an Artificial Neural Network (ANN)

✅ Predict customer credit risk

✅ Improve decision-making in lending processes

---

# 🏦 Business Problem

Financial institutions face significant losses due to loan defaults.

Before approving loans, banks need to evaluate:

- Customer credit history
- Financial behavior
- Debt obligations
- Income stability
- Repayment capability

This project helps automate that process using Deep Learning techniques.
